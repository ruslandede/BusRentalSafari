# 🚌 Bus Rent Management System  
Safari Bus Tour & Travel

---

## 📌 Overview
This project focuses on digitizing the bus rental and tour management process at Safari Bus Tour & Travel, which was previously handled manually.

The system is designed to streamline booking, scheduling, fleet management, and transaction tracking to improve operational efficiency and reduce errors.

---

## 🎯 Key Features
- Bus Fleet Management  
- Customer Booking Management  
- Schedule & Trip Management  
- Driver Assignment  
- Payment & Transaction Tracking
